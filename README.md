# gameOfLife
My first version of the game of life in ```C++```

## Technologies
- ```C++```
- ```Makefile```
- ```bash/zsh```

## Structure
- **src** the main directory with all the files to write the game.
- **include** for all the ```hpp``` files the define fonctions, class and others.
- **bin** for the execution files.
